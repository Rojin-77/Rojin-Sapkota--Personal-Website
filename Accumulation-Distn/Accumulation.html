Project Overview

This project automates the collection and analysis of floorsheet data from the Nepalese stock market to track accumulation and distribution patterns for specific symbols through particular brokers.

A large dataset (~90,000 rows per day) containing stock symbol, quantity traded, rate, amount, buyer broker, and seller broker details was extracted from merolagani.com. Multiple days' floorsheet data were combined into a consolidated dataset.

The processed data enables flexible filtering by buyer and/or seller broker numbers, providing:

  • Total traded amount for each stock symbol
  • Amount traded through the selected broker(s)
  • Net buy/sell position (accumulation or distribution) for each symbol via the filtered broker(s)

Tools & Technologies Used:

  • Python libraries: Selenium (for web automation), Pandas (for data processing), BeautifulSoup
  • ChromeDriver in headless mode for reliable scraping
  • Excel: FILTER, SUMIFS, and other formulas for final analysis and reporting
Note: This is a personal practice project designed to demonstrate skills in web data extraction, automation, and financial data analysis. All data is publicly available and used solely for educational purposes.
